ChangeResolution.exe
========
resolution change tool for windows (CUI tool)

usage
========
<pre>
  
  > ChangeResolution.exe
  usage : ChangeResolution.exe [width] [height]
  
      example
          ChangeResolution.exe 1024 768
  
      resolution
           640,  480
           720,  480
           720,  576
           800,  600
          1024,  768
          1152,  864
          1280,  720
          1280,  768
          1280,  800
          1360,  768
          1366,  768
          1440,  900
  
  > ChangeResolution.exe 640 480
  change resolution width=1280, height=720
  success!

</pre>
